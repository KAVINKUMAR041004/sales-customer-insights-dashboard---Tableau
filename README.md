# Sales & Customer Insights Dashboard (Tableau)

## Project Overview
This project analyzes sales performance and customer behavior using Tableau.  
The dashboard provides insights into revenue trends, profit margins, customer segments, and category performance.

## Tools Used
- Tableau
- Excel
- Data Visualization
- Business KPI Analysis

## Key KPIs
- Total Sales
- Total Profit
- Total Orders
- Average Order Value
- Profit Margin

## Dashboard Insights
• Sales trend analysis over multiple years  
• Customer segmentation (Consumer, Corporate, Home Office)  
• Sales vs Profit relationship analysis  
• Top 10 customers by sales  
• Category-wise sales performance  

## Interactive Filters
- Year
- Category
- Region
- Ship Mode

## Business Insights
- Consumer segment contributes the highest sales
- Technology category generates strong revenue
- Sales and profit show a positive correlation
